## Gradle Kotlin DSL

```kotlin
dependencies {
    implementation("org.ton:ton-kotlin:0.2.15")
}
```

## Maven

```xml

<dependency>
    <groupId>org.ton</groupId>
    <artifactId>ton-kotlin-jvm</artifactId>
    <version>0.2.15</version>
</dependency>
```
